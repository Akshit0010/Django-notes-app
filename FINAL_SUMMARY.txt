================================================================================
TROUBLESHOOTING QUICK FIXES
================================================================================

Issue: "Connection refused" accessing http://localhost
Fix:   Wait 60 seconds for MySQL and Django to start
       Check: docker-compose logs web

Issue: "MySQL Connection Error"
Fix:   Check MySQL is running: docker-compose logs db
       Restart: docker-compose restart db

Issue: Admin password forgotten
Fix:   Create new user: docker-compose exec web python manage.py createsuperuser

Issue: Static files not loading (no CSS/icons)
Fix:   Collect again: docker-compose exec web python manage.py collectstatic --noinput
       Restart: docker-compose restart

Issue: Can't access http://localhost (port in use)
Fix:   Edit docker-compose.yml: ports: - "8080:80"
       Access at: http://localhost:8080

For more issues, see DEPLOYMENT.md or README.md
